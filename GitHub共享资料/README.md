# 研究资料共享库

这个仓库用于存放、整理和共享研究相关资料。

## 目录

- `00_索引/`：资料目录、说明、检索表
- `01_数据/`：原始数据、清洗后数据、CSV、Excel、JSON 等
- `02_图表/`：分析图、统计图、可视化结果、导出图片
- `03_文献/`：论文、报告、书籍摘录、参考资料
- `04_截图/`：网页截图、软件截图、过程记录截图
- `05_小研究/`：小专题、阶段性分析、研究笔记
- `06_其他/`：暂时未分类或不适合放入以上分类的文件

## 使用建议

新增资料后，优先放入对应分类，并在 `00_索引/资料索引.md` 中补一条记录，方便以后查找。
